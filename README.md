## Установка

- Склонируйте проект
- Запуск происходит через `docker-compose`, нужно будет собрать билд докера `docker-compose -p <имя проекта> build`
- Запуск контениров `docker-compose -p <имя проекта> up -d`
- Проинсталировать пакетный менеджер. Для этого нужно зайти в контейнер `docker exec -it <имя проекта>-backend-1 bash` и прописать `composer install`
